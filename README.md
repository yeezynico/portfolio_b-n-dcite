# Portfolio

Bienvenue sur le portfolio personnel de Bénédicte! Ce projet présente ses compétences en graphisme et quelques projets réalisés.

## 🔹 Technologies utilisées
- **React** pour la structure et la logique.
- **Tailwind CSS** pour le design et le responsive.
- **JSX** pour la composition des composants.
- **Vite / Create React App** (selon ce que tu utilises) pour le build.

## 🔹 Fonctionnalités principales
- Présentation de projets et expériences.
- Témoignages de clients.
- Responsive design pour mobile et desktop.
- Réservation de créneau de rendez-vous

## 🔹 Installation et lancement
1. Cloner le projet :  
```bash
git clone <URL_DE_TON_REPO>
```

2. Installer les dépendances :
```bash
npm install
```

3. Lancer le projet en mode développement :
```bash
npm run dev
```

## 🔹 Organisation des fichiers
src/
 ├─ assets/      # Images et médias
 ├─ components/  # Composants réutilisables
 ├─ pages/       # Pages principales
